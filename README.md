# Adverb Parameterized Skills

## Setup

Create environment:
- `conda env create -f environment.yml`
- `conda activate Param_Skills`

Update environment:
- `conda env remove --name Param_Skills`
- `conda env create -f environment.yml`
- `conda activate Param_Skills`

Deactivate environment:
- `conda deactivate`

FFmpeg:
- `brew install ffmpeg`
